Installation des CcdAddins

Visual Studio 2008 starten und unter
Menü->Extras->Optionen->Umgebung->AddIn/Makrosicherheit 
die Standard AddIn-Ordner prüfen und bei Bedarf ein dediziertes
Verzeichnis für das CcdAddIn hinzufügen. Danach Visual Studio 2008 beenden.

Den Inhalt von CcdAddIn_r10.zip entpacken und in 
einen der AddIn-Ordner kopieren.

Visual Studio 2008 starten und unter
Menü->Extras->Add-In-Manager 
das CcdAddIn anhaken.

Ändern des aktuellen Grades

{Add-In-Pfad}\CcdConfig.xml 

bearbeiten.

Das Attribut "aktiverGrad" im Node "Konfiguration" entsprechend ändern.
Der Wert für "aktivierGrad" ist dem Attribut "farbe" im entsprechenden Node "Grad"
zu entnehmen.

Über das Attribut "rgb" kann die Grad-Farbe den eigenen Wünschen angepasst werden.

Bekannte Probleme:

Das CcdAddIn r10 wurde im bewußt im schwarzen Grad entwickelt, um mit dem AddIn ein Beispiel für die Veränderung von Software mit fortschreitenden Graden zu schaffen.
